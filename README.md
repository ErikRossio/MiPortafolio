# Portafolio – Diseño gráfico & edición de video

Landing page minimalista y oscura para mostrar trabajos de diseño gráfico y edición de video, organizada por categorías: **streamers**, **emprendimientos** y **videos TikTok**.

## Estructura

- `index.html` – Maquetado principal de la landing.
- `styles.css` – Estilos, layout, paleta oscura y animaciones suaves.
- `script.js` – Filtros de categorías, modales de proyectos y scroll suave.

## Cómo usar

1. Abre `index.html` en tu navegador.
2. Edita los textos (nombre, redes, descripciones) directamente en el HTML.
3. Reemplaza los bloques de “skeleton” por capturas reales de tus trabajos (imágenes o videos incrustados).

La página es responsive y está pensada para funcionar como una **landing simple** que puedas subir fácilmente (por ejemplo, a GitHub Pages o Netlify).
